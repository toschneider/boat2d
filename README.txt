Implementieren Sie zunächst die Exceptions aus Aufgabe 1. Es empfiehlt
sich, dafür ein eigenes kleines Testprogramm außerhalb dieses
Programmgerüsts zu programmieren, dass die Implementierung testet.

Implementieren Sie dann die Template-Vektorklasse mit den vorgegebenen
Funtionen sowie die typedefs für Integer- und Float-Vektoren. Wenn Sie
sich an die Vorgaben halten, sollte das Programmgerüst
kompilieren. Danach können Sie die Physik für Aufgabe 3 unter
Zuhilfenahme des Vektor-Templates integrieren.
